## Getting Started

1. Clone or download the project
2. Open `index.html` in your web browser
3. Click "Get Started" to access the main application
4. Navigate between pages using the navigation menu

## Features in Detail

### Encoding
Enter a message and use the encoding tool to encrypt it securely. The encoded message can then be shared with others.

### Decoding
Have an encrypted message? Use the decoding tool to decrypt it and read the original message.

### Responsive Navigation
- Desktop users get a full navigation bar
- Mobile users get a hamburger menu for easy navigation
